## Online Store Sales Data Analysis

This project focuses on analyzing sales data from an online store using Python and pandas.  
The goal is to clean the dataset and perform exploratory data analysis to understand sales performance and trends.

### Tasks Performed
- Loaded and explored the online store sales dataset using pandas
- Cleaned the data by handling missing values and correcting data types
- Performed exploratory analysis using summary statistics and grouping
- Analyzed sales, orders, and revenue-related patterns
- Prepared the dataset for further analysis and reporting

### Tools Used
- Python
- Pandas
